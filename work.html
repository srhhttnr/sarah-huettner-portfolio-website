HTML Semantics

In exercise 1.3, I used correct HTML semantics to create efficient and maintainable HTML code, such as comments, SVG's instead of pixelated images, and when to use relative vs. absolute paths.

Web Accessibility

In Exercise 1.4, I checked the accessibility of my webpage by using tota11y, and implemented accessible coding practices such as accessible data tables, using the alt attribute for images, and using ARIA attributes.

CSS Layout

In Exercises 1.5 and 1.6, I used CSS to design and implement different layouts to my webpage, such as the CSS box model, CSS flexbox, and CSS grid.

Responsive Design, Transitions, and Animations

In Exercise 1.7, I used media queries to adjust my design to respond to the screen size of the device on which it is being displayed. I also learned how to enhance vector images with transitions and animations.

CSS Preprocessors and CSS Variables

In Exercise 1.8, I used the CSS Preprocessor sass to implement variables, nesting, and inheritance into my CSS code once it was transpiled. I also used native CSS variables that did not require transpiling.
